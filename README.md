# 🐋 Orca / Portfolio

制作・研究をまとめたポートフォリオサイトです。  
Unity / Unreal / Kotlin / Python / Web など、これまでの制作実績と研究概要を掲載しています。

🌐 **公開サイト:** [https://my-portfolio-liard-delta-81.vercel.app/](https://my-portfolio-liard-delta-81.vercel.app/)

---

### 💻 主要コンテンツ
- 🎮 **Works:** 開発したゲーム・アプリ一覧  
- 🔬 **Research:** 自動運転HMI研究（Unityシミュレータ）  
- ⚙️ **Tech Stack:** 使用言語・スキル一覧  
- 📫 **Contact:** GitHubリンクなど

### 📁 構成
- `index.html`: サイト本体の入口
- `asset/css/`: `base.css` と `components.css` に分割したスタイル
- `asset/js/`: `app.js` を入口に、`solar-system.js` と `works-data.js` を分離
- `source/`: 作品サムネイルや研究画像
- `webGLbuild/`: Unity WebGL の別ビルド
